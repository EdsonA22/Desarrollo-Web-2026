# SINAP - Desarrollo Web 2026

Sistema de Identificacion de Necesidades y Ajustes Pedagogicos.

## Entregable Sprint 1

- Login simulado y registro local de perfiles por rol: estudiante, docente y administrador.
- Registro de necesidades con persistencia en `localStorage`.
- Pantalla de perfil para estudiante.
- Motor basico de recomendaciones por reglas.
- Perfil docente con listado de estudiantes asignados, modal de ajustes razonables y opcion de imprimir/guardar PDF.
- Panel administrador con base de datos de estudiantes, base de datos de docentes, KPIs, graficas CSS y monitor WCAG.
- Interfaz responsive con contraste alto, labels visibles y navegacion por teclado.
